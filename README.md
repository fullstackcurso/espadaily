# EspaDaily

**EspaDaily** es un potente addon para Kodi que combina la navegación por catálogos de TV española con un buscador universal de vídeo.
Funciona como un agregador que permite explorar la estructura de plataformas como **Atresplayer**, **RTVE Play** y **Mediaset Infinity**, buscando automáticamente enlaces públicos disponibles en internet para cada contenido.

![Fanart](plugin.video.espadaily/fanart.png)

## Características Principales

*   **Explorador Unificado**: Navega por Series, Programas, Documentales e Informativos de las principales cadenas de España desde una sola interfaz.
*   **Buscador Universal**: ¿No encuentras lo que buscas en los menús? Usa la búsqueda manual para localizar películas, series internacionales o cualquier vídeo en la red.
*   **Integración con Palantir**: Conecta tus búsquedas con Palantir 3 para ampliar resultados.
*   **Instantáneas y Backups**: Crea copias de seguridad de los menús y de tu configuración para portabilidad total.
*   **Integración de Caché Híbrida**: Sistema inteligente que guarda los menús en tu dispositivo para una navegación ultra rápida, sin esperas de carga.
*   **Gestión de Favoritos e Historial**: Guarda tus series y programas preferidos para acceder rápidamente.
*   **Descargas**: Opción para descargar contenido y verlo offline (cuando el servidor lo permite).

## Instalación (Recomendada)

Para recibir actualizaciones automáticas, se recomienda instalar primero el **Repositorio**:

1.  Descarga el archivo del repositorio: [repository.espadaily-1.0.0.zip](https://github.com/fullstackcurso/espadaily/raw/main/repository.espadaily-1.0.0.zip).
2.  Abre **Kodi** y ve a la sección **Add-ons**.
3.  Selecciona el icono de la caja abierta (arriba a la izquierda).
4.  Elige **"Instalar desde un archivo ZIP"** y selecciona el archivo `repository.espadaily-1.0.0.zip` que acabas de descargar.
5.  Espera al aviso de "Add-on instalado".
6.  Ahora selecciona **"Instalar desde repositorio"**.
7.  Busca **EspaDaily Repository** -> **Add-ons de vídeo** -> **EspaDaily**.
8.  Pulsa en **Instalar**.

## Configuración Recomendada

Para obtener la mejor experiencia, ve a **Opciones -> Contexto Completo**:
*   **Nivel 1 (Recomendado)**: Mejora la precisión de búsqueda combinando "Programa + Título".
